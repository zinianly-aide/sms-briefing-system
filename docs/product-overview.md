# 产品概览

## 项目目标
建设一套企业内部简讯/短信发送系统，支持模板化、群组化、预约发送、历史复制发送、记录查询与统计，并兼容旧系统 add_view / view / show_list 等操作逻辑。

## 核心功能范围
1. 首页 Dashboard
   - 最近 5 次发送记录
   - 快捷发送入口
   - 复制历史发送入口
2. 添加/编辑简讯
   - 简讯状态、厂区群组、群组分类、预约时间
   - 灾害类别、级别、人员属性
   - 简讯内容（一/二）
   - 通知群组、通知单位、群组/单位人员
   - 其他寄送对象、指定电话、说明
3. 简讯详情
   - 只读展示简讯信息、通知对象、发送记录
   - 支持强制取消未完全发送任务
4. 发送记录查询
   - 查询任务与接收人明细
   - 支持状态、时间、模板、发送方式筛选
5. 通讯录管理
   - 联系人维护、私人号码维护、导入导出、HR 同步
6. 群组管理
   - 群组与成员维护，支持外部号码
7. 模板管理
   - 模板维护、默认群组绑定、灾害类别/级别配置
8. 统计分析
   - 发送次数、成功率、模板使用情况、群组触达人数

## 旧功能映射
- add_view -> 新增/编辑简讯页 + 发送预览 + 创建发送任务
- view -> 简讯详情页 + 发送详情 + 接收人明细
- select_emps / select_orgs -> 员工/组织选择器
- show_list -> 发送记录明细表

## 主要角色
- 短信发送员
- 配置管理员
- 通讯录管理员
- 审计/主管
