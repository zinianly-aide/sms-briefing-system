# 页面与流程说明

## 页面清单
- Dashboard 首页
- 简讯新增/编辑页
- 简讯详情页
- 发送记录列表页
- 模板管理页
- 群组管理页
- 通讯录页

## 核心流程
### 1. 即时发送
选择模板/对象 -> 编辑内容 -> 预览接收人 -> 二次确认 -> 创建任务 -> 调用短信网关 -> 回写状态

### 2. 预约发送
填写预约时间 -> 保存简讯和任务 -> 写入调度 -> 到点发送 -> 回写状态

### 3. 复制历史发送
从首页或记录页点击复制 -> 带入历史快照 -> 校验接收人 -> 重新发送

### 4. HR 同步
定时从 HR 拉取人员资料 -> 更新联系人主数据 -> 保留私人号码 -> 记录同步日志

## 页面设计原则
- 企业中后台风格，现代、清晰、简洁
- 重要状态颜色统一
- 发送前必须有预览和二次确认
- 查询页优先可筛选、可回溯
