# Smoke Test Checklist

## 环境准备
1. 启动 MySQL
   ```bash
   docker compose up -d mysql
   ```
2. 启动后端
   ```bash
   cd backend
   mvn spring-boot:run
   ```
3. 启动前端
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## 基础验证
- [ ] 打开 `http://localhost:5173`
- [ ] 默认进入“运营看板”
- [ ] 看板能看到真实联系人/群组/模板/任务统计

## 联系人模块
- [ ] 进入“联系人管理”
- [ ] 列表可加载
- [ ] 可新建联系人
- [ ] 可编辑联系人
- [ ] 可删除联系人
- [ ] 搜索正常

## 群组模块
- [ ] 进入“群组管理”

## 模板模块

## Briefing 模块
