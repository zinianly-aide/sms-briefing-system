# 短信系统开发计划

## 技术栈
- Frontend: React + Ant Design + Vite
- Backend: Spring Boot + MyBatis + MySQL
- Docs: Markdown + Mermaid

## 阶段计划
1. 阶段1：需求与设计 ✅
2. 阶段2：基础工程与仓库初始化 ✅
3. 阶段3：通讯录 / 群组 / 模板 🚧
4. 阶段4：简讯录入与预览
5. 阶段5：发送任务与预约发送
6. 阶段6：查询、详情、复制发送
7. 阶段7：HR同步 / 网关联调 / 审计统计
